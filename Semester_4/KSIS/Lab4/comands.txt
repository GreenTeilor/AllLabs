===1===
login //Аутентифицировать пользователя
Input mail address: ksistest23@gmail.com //Адрес почтового ящика
Input password: [session_key] //Сессионный ключ(можно найти следующим образом: управление аккаунтом Google -> безопасность ->
													двухэтапная аутентификация(включаем, если отключена) -> пароли приложений -> создаем новый и копируем сюда)
===1===

===2===
fetch header|body all|x:y //Прочитать письма пользователя, header - вывод заголовков писем, body - вывод заголовков+содержимого писем, all - вывести все письма, x:y - вывести с письма x по письмо y
Input mailbox: [mailbox] //mailbox - почтовый ящик(например INBOX для входящих писем
===2===

===3===
select [mailbox] //Выбрать почтовый ящик, mailbox - почтовый ящик
===3===

===4===
list [reference][mailbox] //Список всех почтовых ящиков клиента, reference - каталог, где находятся почтовые ящики(если пустой - "", 
								то то почтовые ящики находятся в рабочем каталоге пользователя $HOME), mailbox - почтовый ящик
===4===

===5===
close //Закрывает почтовый ящик(почему-то пишет BAD CLOSE NOT ALLOWED NOW)
===5===

===6===
logout //Закрывает почтовый ящик
===6===
