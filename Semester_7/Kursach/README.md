Курсовой проект по предмету: СПП(Современные платформы программирования).

Веб-приложение "Игра в шашки".
Функционал:
 - Игра с компьютером с возможностью выбрать уровень сложности;
 - Игра по сети;
 - Просмотр информации о шашках и приложении.
